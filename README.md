# 🔞 HentaiHub

Видеоплатформа с возможностью загрузки видео, лайками, комментариями, подписками и историей просмотров.

## ✨ Возможности

- 🔐 Регистрация и авторизация
- 🎬 Загрузка видео с автоматической генерацией превью
- 👍 Система лайков/дизлайков
- 💬 Комментарии к видео
- 🔔 Подписка на каналы
- 📜 История просмотров
- 🔍 Поиск видео и авторов
- 🖼️ Кастомные аватары пользователей
- 📱 Адаптивный дизайн

## 🛠️ Технологии

- HTML5, CSS3, JavaScript
- IndexedDB для хранения видеофайлов
- LocalStorage для пользователей и метаданных
- Canvas API для генерации превью

## 🚀 Как запустить

Просто откройте `index.html` в браузере или перейдите по [ссылке](https://hentaihub2.netlify.app/)

## 📁 Структура

- `index.html` - основной файл приложения
- Все данные хранятся локально в браузере

## 🎯 Особенности

- Видео хранятся в IndexedDB
- Превью генерируются автоматически из видео
- Аватары загружаются с устройства
- Полноценная социальная сеть для видео

## ⚠️ Примечание

Для работы необходима поддержка IndexedDB и localStorage в браузере.
